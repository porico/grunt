grunt
=====

#はじめに

generator-katamariを使用するので、yeomanをインストールする
```
npm install -g yo
npm install -g generator-katamari
```
参考：http://yeoman.io/

プロジェクトのディレクトリで以下のコマンドを実行
```
yo katamari
```

#プロジェクトごとに

```
git clone https://github.com/porico/grunt.git
cd [local directory path]
npm install
grunt
```

※CSSはSmartPhone用
